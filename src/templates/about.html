About

Text Analytica is a free to use scalable cloud application. It was created by Florian Bauer and Nathalie Pett at the University of Bristol in 2018/19 for the module M0010 Cloud Computing. The application is encapsulated into Docker containers run on a Kubernetes cluster and based on Microsoft Azure technologies. For the back-end the Flask framework for Python was used, while the frontend is implemented using Vue.js and Bootstrap.

Whether you are a student, a professional or a scientist: Everyone sometimes needs help by analysing sources, such as notes, papers or books. This current prototype helps you by extracting the text's key terms for an uploaded txt file.
