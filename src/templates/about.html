About

Text Analytica is a free to use scalable cloud application. It was created by Florian Bauer and Nathalie Pett at the University of Bristol in 2018. It is provided by Kubernetes with Dockercontainers using the Oracle Cloud. The backend is the python Flask Framework, in the frontend Vue.js and Bootstrap is used.

Whether you're a student, a proffesional or a scientist: Every one sometimes needs help by finding new sources or books, by getting the keywords from an uploaded file or by getting a short abstract from a long ebook. This Cloud Application should help you now!

Use Text Analytica!
